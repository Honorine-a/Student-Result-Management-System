Project Description

The Student Result Management System is a simple C++ program that accepts student names and marks for three subjects.
It calculates the total, average, and displays whether the student passed or failed.
The project demonstrates C++ concepts such as variables, loops, arrays, functions, conditions, and basic calculations.

Features of the Program

The program can perform the following tasks:

Accept a student’s name
Accept marks for three different subjects
Store marks using arrays
Calculate the total marks
Calculate the average marks
Display whether the student passed or failed
Allow entry of multiple students using loops
Use functions to organize the program
 Display results clearly on the screen
